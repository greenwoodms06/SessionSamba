OPENCONFERENCEPLAN · COMPANION UI KIT · V1

Companion design system

The reference client for the OpenConferencePlan standard. The app owns structure, type and neutrals; each conference owns exactly one accent color (declared in config.json). Track colors belong to sessions, person tints belong to columns — three color channels, never competing.

1 · Color

Neutrals are fixed. The accent is a slot — SIGGRAPH ships indigo here, another conference swaps one value and the whole app rebrands.

6 · Feedback & overlays

Bottom sheets for anything with content (filters, detail, review); centered dialogs only for decisions (tier warning, import confirm); toasts for confirmations. Warnings inform and proceed — they never block.
